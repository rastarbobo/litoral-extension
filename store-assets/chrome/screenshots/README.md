# Chrome Web Store Screenshots

Drop Chrome Web Store listing screenshots here. Chrome requires **1 to 5
screenshots**, recommended **1280x800**, **≤ 16 MB** each, **PNG or JPG**.

Name files `screenshot-01.png` through `screenshot-05.png` (zero-padded).
