# Firefox AMO Screenshots

Drop AMO (addons.mozilla.org) listing screenshots here. AMO accepts **up
to 10 screenshots**, recommended **1280x800 or 1920x1080**, **≤ 4 MB**
each, **PNG or JPG**.

Name files `screenshot-01.png` through `screenshot-10.png` (zero-padded).
